# Filo Clasa 12

Interactive web platform for the Romanian 12th-grade philosophy textbook.

## Overview

A modern web application built with Vite and Tailwind CSS that presents the Romanian 12th-grade philosophy curriculum in an interactive format. Features structured lessons, navigation, and study aids organized by textbook sections.

## Contents

- `src/` -- Source files (Vite + Tailwind)
- `index.html` -- Entry point
- `docs/` -- Additional documentation
- `vite.config.js` / `tailwind.config.js` -- Build configuration

## Getting Started

```bash
npm install
npm run dev
```

Or use `bash run.sh` to start the development server.
